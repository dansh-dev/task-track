Task Tracker, an Angular JS demonstration project.

This project was created to demonstrate my capabilities whilst developing using Angular.

If you have any questions regarding this project please don't hesitate to contact me at: hello@dansh.dev

Any feedback is greatly appreciated and will be noted.

The application has features such as shared projects, easily done via inputting someones user ID into the "share with" field of a project. It also has a task priority system, allowing you to set the priority of a task to low, medium or high. The application also has a task completion system, allowing you to mark a task as complete or incomplete. Thanks for checking it out!

This was developed in 1 day so I apologize for some of the spotty UI elements and lack of features, however I plan to update and improve this project ove the coming days.
